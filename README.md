# Power BI TMDL Documenter 🚀

Este script de Python automatiza la creación de documentación técnica para proyectos de Power BI que utilizan el formato **.pbip (TMDL)**.

## ✨ Características
- 📊 Genera un Excel con múltiples pestañas (Tablas, Medidas, Campos, Hojas).
- 🧹 Filtra elementos estéticos para centrarse en los datos.
- 📂 Compatible con la nueva estructura modular de Power BI Desktop.

## 🚀 Uso rápido
1. Instala las dependencias: `pip install -r requirements.txt`
2. Ejecuta el script: `python documenter.py "ruta/a/tu/proyecto"`
